COME CREARE UN TEMPLATE PER I POST


edit di Mario Rossi- schedule pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi logici la pagina, identificare le parti comuni da applicare a tutte le pagine


le varie aree saranno quindi suddivise in:

  • Contenuti di intestazione del tag
  • Contenuti centrali variabili del tag (corpo principale)
  • Contenuti di fine pagina comuni, da inserire ne l tag

formattazione del codice di testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con CSS. Perciò i passi saranno:

  1. Progettare su carta il markup finale
  2. Scrivere l'HTML dei tag esterni
  3. Applicare gli stili CSS

L'applicazione di stili CSS è cresciuta negli anni sino a raggiunggere la diffusione pressoché totale.
